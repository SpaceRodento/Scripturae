# Bundled Fonts

This directory contains TrueType fonts used by the ABB RAPID Text Generator.

## Fonts Included

- **DejaVuSans.ttf** - DejaVu Sans (regular weight)
- **DejaVuSans-Bold.ttf** - DejaVu Sans Bold
- **DejaVuSansMono.ttf** - DejaVu Sans Mono

## Font Information

**DejaVu Sans**
- Clean, readable sans-serif font
- Good for general-purpose text
- Unicode coverage for many languages

**DejaVu Sans Bold**
- Bold weight variant
- Excellent for titles and headers
- More prominent and easier to see from distance

**DejaVu Sans Mono**
- Monospaced (fixed-width) font
- Good for technical or code-like text
- Each character takes the same width
